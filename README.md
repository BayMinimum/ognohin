# ognohin

Memorizing Japanese words &amp; expression

## [Basic Expressions](#basic-expressions)

- おはよう(ございます): 아침 인사
- こんにちは: 점심 인사
- こんばんは: 저녁 인사
- いただきます: 잘 먹겠습니다
- ごちそうさまでした: 잘 먹었습니다
- いってきます: 다녀오겠습니다
- いってらっしゃい: 다녀오세요
- ひさしぶりです: 오랜만입니다
- すみません: 죄송합니다
- ありがとう(ございます): 감사합니다
- どういたしました: 천만에요
- だいじょうぶです: 괜찮습니다
- じゃ、また: 그럼, 또
- また、あした: 내일 봐요
- また、らいしゅう: 다음주에 봐요
- さようなら: 헤어질때 인사
- ただいま: 다녀왔습니다
- おかえりなさい: 다녀오셨어요?
- おやすみ(なさい): 잘 자요
- おつかれ(さまでした): 수고하셨습니다
